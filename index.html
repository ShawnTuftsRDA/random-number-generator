Frequently Asked Questions

How random is this number generator?
This tool uses JavaScript's Math.random() which generates pseudorandom numbers based on a seed derived from system entropy. It is statistically random enough for games, giveaways, and everyday decisions but is not suitable for cryptographic security purposes.
Can I generate numbers without duplicates?
Yes. In the Multiple tab, select "Not Allowed" in the Duplicates dropdown. The tool will ensure each generated number is unique within the range you specify.
What is the maximum range I can use?
You can use any integer range from negative numbers to very large positive numbers. For the "no duplicates" mode, the range must be large enough to accommodate the number of values you request.
What dice are available in the dice roller?
We support D4, D6, D8, D10, D12, D20, D100 (percentile dice), and 2D6 (two six-sided dice). These cover all standard tabletop RPG dice used in games like Dungeons & Dragons.
Can I use this for a giveaway or raffle?
Yes. Enter 1 as the minimum and the number of participants as the maximum. Click generate to pick a winner. Use "no duplicates" mode when picking multiple winners.
